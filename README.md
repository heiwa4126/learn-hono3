# learn-hono3

[Hono の Best Practices の Building a larger application](https://hono.dev/docs/guides/best-practices#building-a-larger-application)
を試してみる。

To install dependencies:

```sh
bun audit
bun install
```

To run:

```sh
bun run dev
```

open <http://localhost:3000>
